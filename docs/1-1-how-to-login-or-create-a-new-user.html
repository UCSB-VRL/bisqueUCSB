1.1 How to Login or Create a New User

Homepage of BisQue. The landing page of BisQue enables users to interact with sample datasets and view the current analysis modules implemented today. The top right hand corner shows a drop down to Login or Register New User.

Login or Create an Account. After clicking on one of the two options from the dropdown menu above, you will be directed to one of two pages.
